# Ulauncher LibreWolf Profiles

An [Ulauncher](https://ulauncher.io) extension to open LibreWolf with an user profile.

> Inspired by [Ulauncher Chrome Profiles Extension](https://github.com/FloydJohn/ulauncher-chrome-profiles)

## Screenshot

![Preview](./images/screenshot.png)

## Install

1. Open Ulauncher preferences
2. Enter "Extensions" Tab
3. Click "Add extension" on the left
4. Paste the url: `https://github.com/creusvictor/ulauncher-librewolf-profiles`
5. Press "Add"

## Dependencies

You need to create the profiles first inside the `about:profiles` page in LibreWolf. Just enter this url in the address bar and manage your profiles.

Here is a more detailed version on the [LibreWolf official documentation](https://librewolf.net/docs/faq/)

## Configuration

-  **LibreWolf Profiles Keyword**: The keyword to activate the extension. Default: `lw`.

-  **LibreWolf Configuration Folder**: LibreWolf profiles folder. You can find it inside the _"about:profiles"_ page. Default: `~/.config/librewolf/librewolf/`

-  **LibreWolf Executable**: The command line to start LibreWolf. Usually _"librewolf"_. Default `librewolf`.
